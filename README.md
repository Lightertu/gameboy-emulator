# Gameboy Emulator
A gameboy emulator implemented in C/C++ using SDL2 (Only works on Linux so far)

HOW TO USE: 
1. Install SDL2 and make sure it is in your path
2. change the ROM file name in main.cpp. NOTE: it has to be a .txt file
2. make
3. ./main
