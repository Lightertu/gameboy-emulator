# Gameboy Emulator
A gameboy emulator implemented in C/C++ using SDL2 (Only works on Linux so far)

HOW TO USE: 
1. Install SDL2 and make sure it is in your path__
2. change the ROM file name in main.cpp. NOTE: it has to be a .txt file__
2. make__
3. ./main__
